# clash-royale-matchup-calculator
Web application to estimate win probabilities between two Clash Royale decks using match data and machine learning.

## Project Status

Data Collection & Preprocessing (Work in Progress)

## Goal

The goal of this project is to build a system that:

- Collects and processes Clash Royale match data
- Estimates win probabilities between two decks
- Provides a simple web interface for users to compare decks
- (Future) Suggests optimized deck improvements

## How It Works (Planned)
1. Fetch match data from the Clash Royale API
2. Process and normalize deck data
3. Train a machine learning model on historical matchups
4. Predict win probabilities for custom deck inputs

## Tech Stack
- Python (data collection & ML)
- Clash Royale API
- (Planned) backend & frontend
